HOW IT WORKS

Three steps to clean code.

01

Install

One brew command. No dependencies, no config files, no setup wizard.

brew install rinse
02

Run

Point Rinse at any directory. It scans every file and flags AI-generated boilerplate.

rinse .
03

Ship

Commit the diff. Your codebase is clean. Reviewers read code, not AI filler.

git commit -m "clean"
